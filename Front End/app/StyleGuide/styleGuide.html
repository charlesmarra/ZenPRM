Fonts

Body Font: Source Sans Pro

Source Sans Pro is a Google Font. It should be used for all text by default, excluding some headings.

font-family: 'Source Sans Pro', 'Helvetica', 'Arial', serif;

Icons

All icons should be drawn from the Font Awesome icon font library. For consistency, please avoid using Bootstrap's glyphicons.
